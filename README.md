# Siva Prakash S - Portfolio

This is a personal portfolio website designed for placement opportunities.

## How to Use
1.  **Open `index.html`** in your web browser to view the portfolio.
2.  **Customize Content**: Edit the `index.html` file to update your projects, skills, and contact information.
3.  **Styles**: Modify `style.css` to change colors or layout.
4.  **Add Your Photo**: Put your image in the same folder as `index.html` and name it `profile.jpg`.

## Structure
-   `index.html`: Main HTML file containing the content.
-   `style.css`: CSS file for styling and responsive design.
-   `script.js`: JavaScript file for mobile menu and smooth scrolling.
